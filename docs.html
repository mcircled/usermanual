Getting Started

To get start first of all you need to have LMS credentials.

Admin Interface

To get start follow the following steps

  • Login to the LMS
    • go to your lms website example.com/login
    • provide the login details and press the login button
  • Go to the Admin Panel
    • you can navigate from the left sidebar
  • Navigate throug the left sidebar
  • Select the required option from the left sidebar like Campuses, Teachers, Students, etc.
  • Click on the required option from the left sidebar like Add Campus, Add Teacher, Add Student, etc.
  • Fill the required details and click on the submit button
  • You will get the success message
  • Now you can view the details of the added campus, teacher, student, etc.
